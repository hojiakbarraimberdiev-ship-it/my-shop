## 🚀 Запуск

### Локально (без сервера)

Откройте `index.html` в браузере. Но из-за ES6-модулей (type="module") нужен локальный сервер:

```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .

# VS Code: Live Server extension
```

Затем откройте: `http://localhost:8080`

## ✨ Функциональность

### Каталог
- Фильтры: тип (ПК/Ноутбук), состояние (новый/б/у), бренд, цена
- Поиск с debounce (350ms)
- Сортировка по цене и популярности
- Пагинация (12 товаров на страницу)
- Анимация карточек через IntersectionObserver

### Страница товара
- Галерея 4–5 фото с превью
- Увеличение фото (zoom-modal)
- JSON-LD Schema.org
- Характеристики (таблица)
- Похожие товары
- Управление количеством

### Корзина
- Добавление/удаление/изменение количества
- Автоподсчёт суммы
- Сохранение в localStorage
- Счётчик в header

### Оформление заказа
- 3-шаговая форма (данные → доставка → оплата)
- Телефон с кодом Кыргызстана +996
- Валидация всех полей
- Курьер/самовывоз (влияет на стоимость)
- Сохранение заказа
- Toast-уведомление + модальное окно подтверждения

### Авторизация
- Регистрация и вход
- localStorage (без серверной части)
- Защита страниц admin.html и profile.html

### Личный кабинет
- Редактирование данных профиля
- История заказов со статусами
- Смена пароля

### Админ-панель
- CRUD товаров
- Управление статусами заказов
- Статистика (товары, заказы, пользователи, выручка)

## 🎨 Дизайн

- **Стиль**: Tech-style (тёмный/серый/синий)
- **Шрифты**: Syne (заголовки) + JetBrains Mono (цифры, код)
- **Тема**: переключатель светлой/тёмной темы
- **Mobile-first**: адаптивная верстка
- **Анимации**: Apple-style плавные переходы
- **Skeleton loading** при загрузке данных

---

## 🛠 Технологии

- HTML5 + семантическая разметка
- CSS3 (CSS Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6 Modules, async/await)
- localStorage для хранения данных
- IntersectionObserver API
- No frameworks, no build tools

---

## 📱 SEO & Performance

- Meta-теги, Open Graph
- JSON-LD Schema.org (Store, Product, ContactPage, AboutPage)
- `loading="lazy"` для изображений
- Debounce на поиск
- CSS-анимации (не JS)
- Минимизация DOM-перерисовок через innerHTML batch-обновление
- Кэширование данных в памяти
